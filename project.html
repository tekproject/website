LMS Application

This project mainly focus on the core concept of managing leaves that an employee would surely use when he/she works for an organisations, This django project helps the organisation to have different roles to approve or reject an leave applied by the employee. It also has many security protections such as CSRF, ClickJacking and etc.,

Other general web security such as Admin can only add or delete user account, User with the particular permission can only view the particular sites, Manager roled user can view and respond to the leave request made by the employee. Leave request made by the Manager can be responded by the admin.
